Code

int sum = 0;

for(int i = 1; i <= 3; i++) {

    sum += i;

}

 System.out.println("The sum value is: " + sum);

介紹

點擊下方 開始介紹 在for迴圈外面先宣告了一個整數變數，叫做sum，使其等於0
碰到for，執行最左邊的內容：
宣告一個整數變數為i，使其等於1
再來就是中間這一塊：
如果 i <= 3 這條式子成立的話，
執行body的內容，sum += i (也就是sum = sum + i)，
沒有其他code了，
回到最右邊這一格，i++ (相當於i = i + 1)
 再經由中間這一個判斷式來決定是否要執行body，i現在是2，還是小於等於3，成立，執行body
再經由中間這一個判斷式來決定是否要執行body，i現在是3，還是小於等於3，成立，執行body
再經由中間這一個判斷式來決定是否要執行body，i現在是4，大於3，原式(i <= 3)不成立，不執行body，離開for
最後印出sum的值，結束程式。
